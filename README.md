
# Zadania – podstawy programowania Python i C#

Repozytorium zawiera rozwiązania trzech zadań programistycznych
zrealizowanych jako aplikacje konsolowe w językach **Python** oraz **C#**

## Zawartość projektu

### Python (python/main.py)
Program uruchamia się z menu, które umożliwia wybór jednego z trzech zadań:

1. **Kalkulator dwóch liczb**  
   Można wybrać: dodawanie, odejmowanie, mnożenie oraz dzielenie

2. **Konwerter temperatur (Celsjusz i Fahrenheit)**  
   Przeliczanie temperatur zgodnie z podanymi wzorami

3. **Średnia ocen ucznia**  
   Obliczanie średniej z podanych ocen i sprawdzenie zaliczenia przedmiotu



### C# (csharp/Program.cs)
Program w języku C# realizuje te same trzy zadania co wersja Pythonowa, zawiera menu
oraz kodowanie UTF-8 w celu poprawnego wyświetlania polskich znaków w konsoli

---

## Uruchamianie programu

### Python
- Uruchomić plik `main.py` w środowisku obsługującym Pythona 3  
- Program działa jako aplikacja konsolowa

### C#
- Uruchomić plik `Program.cs` jako aplikację konsolową (.NET)
